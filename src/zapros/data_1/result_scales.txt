Шкала для критериев Новизна и Глубина : Новый , Проработанный -> Средняя новизна -> Не новый -> Средне проработанный -> Не проработанный;
Шкала для критериев Новизна и Влияние : Проработанный , Большое влияние -> Среднее влияние -> Не влияет -> Не проработанный -> Средне проработанный;
Шкала для критериев Глубина и Влияние : Новый , Большое влияние -> Среднее влияние -> Средняя новизна -> Не влияет -> Не новый;
Единая Порядковая Шкала: Новый , Проработанный , Большое влияние -> Среднее влияние -> Средняя новизна -> Не влияет -> Не новый -> Не проработанный -> Средне проработанный;
